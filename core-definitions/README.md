# The Astral Network

The `Astral Network` is a network of `Identities` that communicate by
sending `Queries` to each other, which, if accepted, open `Channels` used
to exchange data in the form of `Objects` and `Object Streams`.

The files in this directory define the core primitives as abstract,
language-independent types. For how these primitives compose at runtime,
see [Network Architecture](../topics/network-architecture.md). All files
together total ~170 lines.
