# Object

* An `Object` represents immutable data on the `Astral Network`.
* An `Object` consists of an optional [`Object Type`](object-type.md) and an optional `Payload`.
* An `Object` with no `Object Type` is called an `Untyped Object`.
* An `Object` with a type is called a `Typed Object`.
* A `Payload` is any binary data.
* Every `Object` has an identifier called the [`Object ID`](object-id.md).
* An `Empty Object` is a special `Object` with no `Object Type` or `Payload`.
