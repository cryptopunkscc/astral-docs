# Node

* A `Node` is an active agent on the `Astral Network`.
* A `Node` can establish a `Link` to another `Node` over an `Exonet`.
* A `Node` can host `Apps`.
* A `Node` relays `Queries` and `Channel` data for the `Apps` it hosts.
* A `Node` assigns `Identities` to the `Apps` it hosts that are valid until
  the `App` is removed from the `Node`.
* A `Node` manages the cryptographic keys of the `Apps` it hosts.
* A `Node` and its `Apps` produce signed `Objects` that can be used to prove
  to other `Identities` that the `Node` hosts those `Apps`.
