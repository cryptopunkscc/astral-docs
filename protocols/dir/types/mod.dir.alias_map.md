# mod.dir.alias_map

A snapshot map from alias strings to identities.

## Fields

* Aliases (object) – Map from alias name (`string16` key) to the corresponding `identity` value.

## Example

```json
{
  "Type": "mod.dir.alias_map",
  "Object": {
    "Aliases": {
      "alice": "02bef8840eb35ef2ae3c83c07cb5779278904f99cb4103f71e37cc69931ae5e15f",
      "bob": "037f990e61acee8a7697966afd29dd88f3b1f8a7b14d625c4f8742bd952003a590"
    }
  }
}
```
